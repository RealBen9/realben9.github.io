Rules

Draw bridges between each circled island so that the number of bridges leaving each island are the same as the number written inside the circle. There cannot be more than two bridges between any pair of islands. It must be possible to walk from one island to any other island via the bridges in the finished puzzle, therefore no area of the puzzle can be isolated from any others. Bridges must be horizontal or vertical only, and cannot cross each other.
