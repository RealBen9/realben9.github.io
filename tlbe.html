Board Editor

Any boards that you make here can be used in TLU's zen mode. Just remember to copy the board string first. (May connect to TLO via cookies)

Width :

How to use

Enter numbers into the board string input, then press "View" to view the output. This output is functionally identical to the output of TLO.

A "board string" is intended to have up to 200 characters, where each character represents 1 mino. (The 1st character is the bottom-left mino & the 2nd is the mino to the right.)

Each character can be a number from 0-9, which represent the following : 0 : Red | 1 : Orange | 2 : Yellow | 3 : Green | 4 : Light Blue | 5 : Blue | 6 : Purple | 7 : White | 8 : Black | 9 : Empty

Note that any character that isn't 0,1,2,3,4,5,6,7 or 8 will act as a 9. Also note that any trailing "9"s can be truncated.
